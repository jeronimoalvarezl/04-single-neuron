# Single Neuron Model – Vitamin Deficiency Dataset

## Descripción

Este proyecto entrena un modelo de una sola neuro para predecir si una persona tiene deficiencia de vitaminas o no, esto a partir de variables clínicas y de estilo de vida.

## Requisitos

Instalar las siguientes librerías de Python:

```
pandas
numpy
matplotlib
seaborn
scikit-learn
tensorflow
```

## Ejecutar el notebook

1. Clonar el repositorio:

```
git clone https://github.com/jeronimoalvarezl/04-single-neuron
cd https://github.com/jeronimoalvarezl/04-single-neuron
```

2. Asegurarse de que el dataset esté en la misma carpeta del notebook.

3. Ejecutar todas las celdas del notebook en orden.

## Dataset

Vitamin Deficiency Disease Prediction Dataset
https://www.kaggle.com/datasets/nudratabbas/vitamin-deficiency-disease-prediction-dataset
